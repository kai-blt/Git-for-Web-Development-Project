/*Questions*/
1. What is Semantic HTML? 
Special HTML tags that provide meaning for screen readers, Google bots, etc.

2. What is HTML used for? 
HTML is the structure for a website

3. What is an attribute and where do we put it? 
Attributes are placed within HTML tags to add data settings to an element such as id, name, src, etc.

4. What is the h1 tag used for? How many times should I use it on a page?
H1 is a heading tag, it should only be used once per page. It gives heirarchy and meaning for SEO, Google bots, etc.

5. Name two tags that have required attributes
<a><img>

6. What do we put in the head of our HTML document? 
Page title, CSS includes, Open Font (Google), Meta Tags.

7. What is an id?
id is an element selector. Must be unique.

8. What elements can I add an id to? 
You can add an id to any element

9. How many times can I use the same id on a page? 
Only once, they must be unique

10. What is a class? 
Classes can be used by multiple elements. They're used in CSS to provide style to a certain element.

11. What elements can I add a class to? 
You can add a class to any element

12. How many times can I use the same class on a page? 
You can use the same class multiple times. 

13. How do I get my link to open in a new tab?
target="_blank"

14. What is the alt attribute in the image tag used for? 
The alt attribute is used for screen readers and for Google bots that crawl websites for rankings. 
Also, if you image is broken, the alt text will be displayed which is best practice.

15. How do I reference an id?
You reference it with the # symbol

16. What is the difference between a section and a div
Section is a semantic tag which provides meaning for screen readers and is more human understandable in code.
Div should be used more for container elements that can be moved around the page. 

17. What is CSS used for? 
Cascading Style Sheet - used to style a website.

18. How to we select an element? Example - every h2 on the page
In CSS you would just type the tag name followed by {}. For example:
h2 {
    /*insert attributes here*/
}

19. What is the difference between a class and an id? - Give me an example of when I might use each one
id is used to select a specific element. You can use this to jump around the page via anchor tags and referencing
an elements specific id with #id-name. Classes are great for styling many elements at once and writing compact,
performant code. Specify once in CSS, change multiple elements in HTML.

20. How do we select classes in CSS?
use the . selector. (.classname). For example:
.classname {
    /*insert attributes here*/
}

21. How do we select a p element with a single class of “human””?
p.human {
    /*insert attributes here*/
}

22. What is a parent child selector? When would this be useful? 
Parent child selector is useful for selecting specific elements nested within other elements. You could also
use nth-child or nth-type selectors to be even further specific in selecting something nested within another element.

23. How do you select all links within a div with the class of sidebar?
div.sidebar a {
    /*insert attributes here*/
} 

24. What is a pseudo selector?
Pseudo selectors are used to modify special states using keywords on specific tags. For example:
a:hover will modify the hover state of an anchor tag given the attributes provided.

25. What do we use the change the spacing between lines?
line-height

26. What do we use to change the spacing between letters?
letter-spacing

27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
text-transform: uppercase, text-transform: lowercase, text-transform: capitalize;

28. How do I add a 1px border around my div that is dotted and black?
div {
    border: 1px dotted #000;
}

29. How do I select everything on the page? 
* {
    /*insert attributes here*/
}

30. How do I write a comment in CSS?
Utilizing /* and */ with a message in between. For example: /* here is a comment */

31. How do I find out what file I am in, when I am using the command line? 
pwd

32. Using the command line - how do I see a list of files/folders in my current folder?
ls, ls -a, or ls -la 

33. How do I remove a file via the command line? Why do I have to be careful with this? 
rm (filename). Must be careful because there is no undo option.

34. Why should I use version control? 
Version control is a great tool to be able to track your changes across many revisions of your work.
It allows you to roll back should there be an issue and also better facilities collaboration amongst teams
working effectively on large projects utilizing the same codebase.

35. How often should I commit to github?
Commit often with meaningful changes.

36. What is the command we would use to push our repo up to github? 
git push -u origin branchname

37. Walk me through Lambda's git flow. 
1. Fork the project you want to work on so as to not disturb the master branch/project.
2. Manage access to add our TL as a contributor to the project.
3. Clone the project locally. Use the terminal to LS to the directory where you'd like to store the project.
4. In the terminal use: git clone (url of project)
5. cd into the project directory: cd (project directory)
6. Create a new branch: git checkout -b firstname-lastname
7. Complete your work and add changes to staging: git add . 
8. Commit changes with a helpful message: git commit -m 'helpfulmessage'
9. Push changes up to GitHub: git push -u origin branchname
10. Submit pull request by navigating to your forked Repo. Change the branch dropdown to the new branch you created earlier (firstname-lastname).
11. Click on the 'New Pull Request' button.
12. Once on the pull request form, be sure to change the base repository to the forked version of the repo.
13. Add our TL to the reviewers, fill out requisite info, and hit 'Submit Pull Request'
14. Await TL to merge into master.
